## Game of War 

War is a simple card game, each player turns up a card at the same time and the player with the higher card takes both cards and puts them, face down, on the bottom of his stack.

If the cards are the same rank, it is War. Each player places 3 cards face down,The 4th card will be place Face up. The player with the higher cards takes both piles (six cards). If the turned-up cards are again the same rank, each player places another card face down and turns another card face up. The player with the higher card takes all 10 cards, and so on.

## Getting Started

Just Run the HTML file and it will pop up in your browser Console.
